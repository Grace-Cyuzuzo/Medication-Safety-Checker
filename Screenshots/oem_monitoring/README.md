# OEM Monitoring Screenshots

This folder contains screenshots from Oracle Enterprise Manager (EM Express) for the Medication Safety Checker project.  
Oracle XE 21c includes a reduced EM Express interface with the following monitoring features:

---

## 1. Database Instance Overview
File: `db_instance_overview.png`  
Shows:
- Database version  
- Instance status  
- Uptime  
- Platform information  

---

## 2. Performance Activity
File: `performance_activity.png`  
Shows:
- Database activity information  
- Services activity  
- Performance charts  

---

## 3. Storage Overview
File: `storage_overview.png`  
Shows:
- Datafile and tablespace usage  
- Space distribution  
- Storage health  

---

These screenshots demonstrate that the database is running correctly and provide basic monitoring information required for the PL/SQL practicum project.
